# Persona Quality Auditor V5

This prompt is reserved for optional offline corpus evaluation. It is not called in the default persona generation path.

Evaluate coherence, lived-scene quality, local grounding, behavioural depth, voice distinctiveness, sensitive-scenario handling, stereotype risk, product-research usefulness, and whether persona core stays distinct from project-specific inference. Compare against a library only to report duplication or panel coverage; never force every person to be unique. Identify concrete weaknesses and preserve the synthetic-only limitation.
