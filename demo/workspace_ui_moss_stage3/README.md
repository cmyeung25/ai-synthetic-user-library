# Workspace UI Moss Stage 3

This prototype advances the selected `Moss Console` direction into the next Workspace UI story:

- intent inference
- progressive clarification
- draft plan contract preview
- explicit execution and evidence boundaries before confirmation

Open `demo/workspace_ui_moss_stage3/index.html` in a browser.

What it demonstrates:

- the system can infer a likely research mode from natural-language intent and supplied artifacts
- clarification can stay minimal and targeted instead of expanding into a large form
- the product can expose a structured draft plan object before a real run is submitted
- unsupported requests such as live-app capture can remain in the same flow while staying explicit about backend gaps
